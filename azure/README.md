# azure-aks

Setup an Azure AKS cluster to run the microservices on.

To create the cluster, run:

```sh
./setup.sh
```

Check the script for details on the cluster size, Kubernetes version, etc.

To delete the cluster, run:

```sh
./teardown.sh
```
